#  E-Commerce Application

A fully functional full-stack e-commerce store layout built using a Node.js/Express backend server and a vanilla JavaScript frontend interface.

##  Project Features
- **Live Inventory Data:** Fetches product details directly from an active Express API.
- **Dynamic Grid Layout:** Clean UI featuring products, prices, and images.
- **Interactive Shopping Cart:** Users can seamlessly add items, calculate totals, and remove items dynamically.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **Backend:** Node.js, Express.js, CORS Middleware

## How to Run Locally

### 1. Start the Backend Server
Navigate into the directory, install dependencies if required, and spin up the Node environment:
\`\`\`bash
node server.js
\`\`\`
The server will boot up and actively monitor requests at `http://localhost:5000`.

### 2. Launch the Web Interface
Simply open the `index.html` file in any modern web browser to browse the catalog, view live layouts, and manage your cart.
